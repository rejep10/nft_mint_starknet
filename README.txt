Установка
пропишите в терминал
pip install starknet.py
pip install loguru

В текстовые файлы address.txt и private_key.txt пропишите адреса и приватный ключ вашего кошелька соотвественно, с каждой строки.

Код минтит нфт https://quantum.starknet-ecosystem.com/ . Так же проверяет наличие нфт на кошельках и записывает их в addresses_with_nft.txt.
На данный момент минт не актуален, а вот чекер да.
В 14 строке кол-во попыток для минта.
В 68 строке кода main.py можете поставить задержку между кошельками.

Запускать main.py для минта, для проверки check_nft.py
